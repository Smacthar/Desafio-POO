**Desafio: Programação Orientada a Objetos**

O objetivo do desafio é utilizar a POO para utilizar na prática uma das principais ferramentas, sendo elas: **Abstração, Encapsulamento, Herança e Poliformismo através do Java.**

O ensinamento foi todo realizado por etapas e de maneira bem explícita, para que todas as dúvidas fossem sanadas e a prática pudesse ser atribuída de maneira clara e sem frustrações.
